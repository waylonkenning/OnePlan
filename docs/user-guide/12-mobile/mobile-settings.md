# Mobile Settings

![Mobile settings bottom sheet open on a phone screen](../../public/features/mobile-settings-sheet.png)

The mobile settings sheet provides quick access to display and filtering options when using Scenia on a phone. It replaces the desktop toolbar controls, which are hidden on mobile viewports.

## Opening and closing the sheet

Tap the settings icon in the top-right corner of the mobile header to open the sheet. To close it, tap the backdrop outside the sheet or press Escape. The sheet slides up from the bottom of the screen.

## Navigating between views

The tab bar at the bottom of the screen switches between the four main views:

- **Visualiser** — Card View of your portfolio
- **Data Manager** — tabular data entry
- **Reports** — initiative and resource reports
- **History** — version history and undo

The active tab is indicated by a border highlight.

## Card View bucket mode

The bucket mode selector controls how initiatives are grouped within each asset card. Choose from:

- **Timeline** — groups by the configured date window
- **Quarter** — groups by calendar quarter of start date
- **Year** — groups by calendar year of start date
- **Programme** — groups by programme name
- **Strategy** — groups by strategic theme

The selected mode is saved to the device and persists across sessions.

## Date range filter

Set a start date and months window to limit which initiatives appear in Card View. Only initiatives whose start date falls within the range are shown. This setting works alongside the bucket mode — you can filter to a specific period and then group the results by quarter or programme.

## Display toggles

Four toggles control what additional information appears in initiative rows within cards:

| Toggle | What it shows |
|---|---|
| Descriptions | Initiative description text below the initiative name |
| Budget | Formatted budget amount (NZD) |
| Relationships | Names of related initiatives and their dependency type |
| Conflicts | Conflict badges in asset card headers |

Turning off Conflicts hides the badge count from card headers. The other three toggles add or remove detail within each initiative row. All four settings persist to the device.
