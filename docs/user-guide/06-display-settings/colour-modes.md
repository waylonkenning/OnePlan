# Colour Modes

![Colour by status](../../public/features/colour-by-status.png)

The **View** popover in the header (click the **View** button) lets you choose how initiative bars are coloured across the timeline canvas.

## By Programme (default)

- Each programme is assigned a colour. All initiatives in that programme use the same colour.
- The floating legend shows each programme name alongside its colour swatch.

## By Strategy

- Each strategy is assigned a colour.
- Useful for seeing how strategic themes are distributed across the portfolio.

## By Progress

- Colours reflect each initiative's current status: Planned (slate), Active (blue), Done (green), Cancelled (red).
- Use this mode to quickly assess portfolio health at a glance.

## Behaviour

- The selected colour mode persists across page reloads.
- The floating legend updates automatically to reflect the active mode.
