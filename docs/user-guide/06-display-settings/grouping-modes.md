# Grouping Modes

![Grouped timeline](../../public/features/grouped.png)

The same **View** popover that controls colour mode also controls how the timeline swimlanes are grouped.

## By Asset (default)

- Initiatives are grouped by IT asset within their category.
- This is the standard portfolio view for day-to-day planning.

## By Programme

- All initiatives belonging to the same programme are grouped together in a single swimlane, regardless of which asset they belong to.
- Useful for programme-level reviews where you want to see all work under a programme in one place.

## By Strategy

- All initiatives belonging to the same strategy are grouped together.
- Useful for strategic alignment reviews where you need to assess coverage and distribution across themes.

## Behaviour

- Switching grouping mode re-draws the swimlane rows with the appropriate labels. All initiative bars remain visible.
- The selected grouping mode persists across page reloads.
