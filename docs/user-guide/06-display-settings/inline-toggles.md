# Inline Display Toggles

![](../../public/features/inline-toggles.png)

Four icon toggle buttons in the header control what detail is shown on the timeline canvas. Each button shows whether it is active or inactive. On mobile, these toggles are available in the settings sheet.

## Conflict Detection

- Shows or hides red conflict markers when two initiatives on the same asset overlap in time.
- Toggle off to reduce visual noise when conflicts are expected or intentional.

## Dependency Lines

- Shows or hides all dependency arrows on the timeline.
- Toggle off for a cleaner view when arrows are obscuring bars.

## Descriptions

- Shows or hides initiative description text on bars and on mobile card rows.
- Toggle on to see notes without opening the edit panel.

## Budget

- Cycles through three states: off → label mode → bar-height mode → off.
- Label mode shows a formatted budget figure directly on the bar.
- Bar-height mode scales bar height proportionally to the budget value, giving an instant visual sense of relative investment across initiatives.

## DTS Adoption Status

- Shows or hides a coloured badge on each DTS asset row header indicating that asset's current adoption status (Not Started, Scoping, In Delivery, Adopted, Decommissioning Incumbent, Not Applicable).
- This toggle is only visible in DTS and Mixed workspaces.
- Set adoption status per asset in **Data Manager → Assets**.

## Behaviour

- All toggle states persist across page reloads.
