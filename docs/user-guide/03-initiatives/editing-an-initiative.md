# Editing an Initiative

*Screenshot: the InitiativePanel slide-in panel open on the right side — to be captured.*

## Opening the panel

Click an initiative bar on the timeline to open the edit panel on the right. The panel shows all fields for that initiative.

## Saving

Click **Save Changes**. Changes are written to IndexedDB immediately.

## Closing

Click the X button, the Cancel button, or press Escape. Unsaved changes are discarded.

## Keyboard navigation

The Tab key cycles focus through all fields within the panel without jumping to elements behind it.

## Editing from the Data Manager

You can also edit all initiative fields in the Initiatives table. See [Data Manager — Inline Editing](../07-data-manager/inline-editing.md).
