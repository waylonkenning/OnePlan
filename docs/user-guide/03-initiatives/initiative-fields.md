# Initiative Fields Reference

Every initiative has the following fields. All are available in both the edit panel and the Data Manager initiatives table.

| Field | Description | Where it appears |
|---|---|---|
| Name | The initiative title | Bar label, reports |
| Asset | The IT asset this initiative belongs to | Swimlane grouping |
| Programme | Portfolio programme alignment | Colour-by-programme legend, reports |
| Strategy | Strategic theme alignment | Colour-by-strategy legend, reports |
| Status | Planned / Active / Done / Cancelled | Colour-by-status mode |
| Start Date | When the initiative begins | Bar left edge |
| End Date | When the initiative ends | Bar right edge |
| Progress % | Percentage complete | Fill overlay on the bar |
| Owner | Primary responsible person (from the Resources list) | Initials badge on the bar |
| Budget | Planned cost | Budget label on bar, Budget Summary report |
| Description | Free-text notes | Tooltip, mobile card rows |
| Application | The application within the asset this work relates to | Metadata only |
| Resources | Team members assigned to this initiative | Names on bar when "Show Resources" toggle is on |

**Owner** and **Resources** are populated from the [Resources roster](../08-resources/resource-roster.md). Add people there before assigning them.
