# Editing Dependencies

*Screenshot: dependency edit panel open — to be captured.*

## Opening the Panel

Click a dependency arrow on the timeline. The dependency edit panel slides in showing both initiative names in full, along with the current type and direction.

## Changing the Type

Select Blocks, Requires, or Related from the type selector in the panel. The arrow updates immediately on the timeline.

## Reversing Direction

Click the Reverse button to swap the source and target of the dependency. Use this to correct a dependency that was drawn in the wrong direction without deleting and redrawing it.

## Deleting a Dependency

Click the trash icon in the panel. A confirmation modal appears before the dependency is removed. This action cannot be undone without using the undo function.

## Repositioning an Arrow

Drag the arrow horizontally on the timeline to move its routing midpoint. This is useful when multiple parallel arrows overlap and are difficult to distinguish or click. The offset persists across page reloads.

## Overlapping Arrows

If two arrows are close together, clicking near them shows a disambiguation popover listing both dependencies. Click the one you want to open in the panel.

## Closing the Panel

Close the panel with Escape or the X button.
