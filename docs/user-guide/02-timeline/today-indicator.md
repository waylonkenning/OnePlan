# Today Indicator

*Screenshot: the timeline canvas showing the red vertical "Today" line — to be captured.*

A red vertical line labelled **"Today"** is permanently rendered at the current date. It gives you an immediate visual reference for where active work stands relative to the plan.

The line updates automatically when you change the timeline start date or duration.

If today falls outside the visible window — for example, when you are looking at a future planning horizon — the line will not be visible. That is expected behaviour.
