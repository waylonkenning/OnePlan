# Configuring the Timeline Window

![](../../public/features/configuring-the-window.png)

Three controls in the header let you define what portion of the planning horizon you are looking at. All three settings persist across page reloads via IndexedDB.

## Start Date

The date input in the header sets where the timeline begins. Change it to jump to any planning horizon. In the 3-month view, weekly columns always start on Monday regardless of the exact date you enter.

## Duration (Months)

The dropdown next to the start date sets how many months are visible. If any initiative extends beyond the selected duration, the timeline auto-extends to fit it — nothing is ever cut off.

## Zoom

The + and - buttons widen or narrow the column widths.

- Use a wider zoom for a detailed week-by-week view.
- Use a narrower zoom for a high-level overview across many months.
- The zoom buttons are disabled at minimum and maximum levels.
- Your zoom level persists across page reloads.
